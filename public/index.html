REST API

A simple REST API with endopints for managing users, income, and expenses.

Home Page

  1. GET / :- A brief description of the REST API and Available Endpoints

Users Endpoints

  1. GET /users - Retrieve all users.
  2. POST /users - Add a new user.
  3. PUT /users/:id - Update an existing user by ID.
  4. DELETE /users/:id - Delete a user by ID.

Expense Endpoints

  1. GET /expenses - Retrieve all expenses.
  2. POST /expenses - Add a new expense.
  3. PUT /expenses/:id - Update an existing expense by ID.
  4. DELETE /expenses/:id - Delete an expense by ID.

Income Endpoints

  1. GET /income - Retrieve all income.
  2. POST /income - Add a new income.
  3. PUT /income/:id - Update an existing income by ID.
  4. DELETE /income/:id - Delete an income by ID.
